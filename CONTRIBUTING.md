# How to Participate

Thanks for your interest in joining the Agent Metadata Specification Working Group.
We’re building the open standard for Agentic Metadata together, and we’re excited to have you involved.

## How to Contribute

Here are the types of contributions we are looking for:
1. Framework Improvements
2. Documentation
3. Research and Analysis
4. Industry-specific agent metadata standards (e.g., healthcare, financial services)

## Getting Started

Here’s how to get started:
- Visit the GitHub Repository https://github.com/TavroOrg/Agent-Metadata-Specification
- Join our Slack community https://tavro.slack.com/archives/C0A8HD7EAJX
- Email us at info@tavro.ai

If you provided a GitHub username, you may receive an invitation to collaborate shortly.

If you have any questions or ideas, feel free to jump into Slack and introduce yourself.

## Questions

- Open an issue for questions
- Tag maintainers if needed
- Check existing documentation first
